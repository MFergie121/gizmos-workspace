Dataset C — BOM Climate Data

Source:
- Bureau of Meteorology climate statistics pages for the station IDs specified in the assignment brief.

Method:
- Used the BOM extended climate statistics pages for each station.
- Extracted the following monthly statistics for March through September:
  - Mean maximum temperature (°C)
  - Mean minimum temperature (°C)
  - Mean rainfall (mm)
- Entered them into a tidy CSV with one row per city-month combination.
